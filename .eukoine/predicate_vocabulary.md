# Predicate vocabulary — family consult-before-coin

**Status:** Authoritative. Part of the `.eukoine/` single source (EKDP-032 / CDP-057):
one canonical copy at the EuKoine hub, synchronized byte-identical into every member.
A mirror is never hand-edited.

**Scope:** the family's **shared code lexicon** — signifiers for classes, fields, JSON
keys, CLI flags, schema names, and technical prose nouns. This is **not** the Greek
glossary (EKDP-017): lexicon ≠ glossary.

**Positive registry:** [`predicate_lexicon.yaml`](predicate_lexicon.yaml)

---

## Obligation (every family repo)

1. **Consult the registry before coining** any code signifier.
2. **Use the canonical term verbatim** when a referent is already baptised.
3. **On correction, delete the wrong word** — never add a synonym beside it.
4. **Cite this path** from member docs; do not restate registry entries elsewhere (G4).

Member-local vocabulary may be minted only for suite-local referents absent from the
registry. Divergent second copies of this file, or of `predicate_lexicon.yaml`, are
drift.
